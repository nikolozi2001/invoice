# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Next.js 16 and React 19
- PostgreSQL database with Prisma ORM
- Landing page with hero section
- Features showcase with Georgian language support
- Waitlist functionality with form validation
- Docker Compose setup for development
- Database seeding with Georgian sample data
- GitHub workflows for CI/CD
- Comprehensive documentation

### Changed
- Upgraded from SQLite to PostgreSQL
- Updated to Prisma 7 with new configuration format

### Security
- Added security policy and vulnerability reporting process
- Implemented input validation for waitlist form

## [0.1.0] - 2026-01-17

### Added
- Initial release
- Landing page for Georgian Invoice SaaS
- Waitlist registration system
- Basic project structure with Next.js App Router
- Tailwind CSS styling system
- Prisma database setup
- Development environment with Docker
